# Cleaner

Remove Metadata from your photos at once from your terminal.

```
\____/_____/_____/_/  |_/_/ |_/_____/_/ |_|  

```

## What it does

- Strips EXIF data from the images in "inputs" folder and output the result in the "outputs" folder

## Installation

Clone the repository:
```bash
git clone https://github.com/vladkotov92/cleaner.git
```

## Usage

After cloned the repository insert the images you want to remove data in the "inputs" folder and then run:
```bash
cd cleaner
python3 cleaner.py
```
